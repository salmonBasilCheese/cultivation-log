# Tech Stack

## Frontend

- **Framework**: React 18+
- **Build Tool**: Vite 5+
- **Language**: JavaScript (ES6+)
- **Styling**: Vanilla CSS (CSS Modules optional, but standard CSS preferred for simplicity)
- **State Management**: React Hooks (useState, useEffect, useContext)
- **Routing**: Internal state-based routing or simple conditional rendering (Keep it simple for MVP)

## Backend (BaaS)

- **Service**: Supabase
- **SDK**: `@supabase/supabase-js`
- **Features**:
  - Auth (Google OAuth)
  - Database (PostgreSQL)
  - Storage (Images)

## Development Tools

- **Node.js**: LTS
- **Package Manager**: npm
- **Version Control**: Git
